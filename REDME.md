Office Network Project

وصف المشروع

هذا المشروع يحاكي شبكة مكتب صغير تحتوي على 16 جهاز كمبيوتر (PCs)، راوتر، سويتش، وسيرفر داخلي. الهدف من المشروع:

توصيل الأجهزة بشكل صحيح عبر سويتش واحد.

ضبط عناوين IP لكل جهاز، Default Gateway، DNS.

اختبار الاتصال بين الأجهزة عبر Ping.

إضافة سيرفر لتقديم الخدمات الأساسية (مثل Web وDNS).

تجربة عمل جدار ناري (Firewall) كخيار إضافي.

اختبارات Ping

تم اختبار الاتصال بين جميع أجهزة الـ PCs → جميع الاختبارات ناجحة.

تم اختبار الاتصال مع السيرفر → ناجح.

يمكن تجربة ping من أي جهاز إلى Default Gateway → ناجح.


ملاحظات

الشبكة تعمل على سويتش واحد بدون VLANs (يمكن إضافتها لاحقًا).

يمكن استخدام Firewall كخيار أمني إضافي.

السيرفر يقدم خدمات أساسية مثل DNS وWeb server.

جميع الأجهزة لديها Default Gateway = 192.168.1.1.
